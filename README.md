# Anagrams

Anagrams is a Maven based Java project who build —Åommand line tool for rebuild singly words in a sentence into anagrams.

## Example

All non letters symbols are stay on the self place

Input:   a1bcd efg!h

Output: d1cba hgf!e

## Usage

```cmd
#Open comand line in root project directory

#Build project
mvn package

#Run
java -jar target\anagrams-1.0.jar

```
