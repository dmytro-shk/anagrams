# Anagrams

Anagrams is a Maven based Java project who include —Åommand line tool for rebuild singly words in a sentence into anagrams.

## Example

All non letters symbols are stay on the self place

Input:   a1bcd efg!h

Output: d1cba hgf!e

## Additional information

- 9 Junit5 tests are included
- Built using the principles of OOP

## Usage

- Open comand line in project root directory

- Build project

```
mvn package
```
- Run

```
java -jar target\anagrams-1.0.jar

```
